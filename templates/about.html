About This Project

This system provides secure image encryption and decryption using Multi-Agent Swarm Behavior (MASB) and Quantum-Inspired Chaotic Encoding (QICE).

Objectives

  • Ensure confidentiality while transmitting digital images
  • Provide a lossless and reversible decryption process
  • Enhance cryptographic strength using swarm intelligence & chaos theory
  • Deliver a fast and user-friendly encryption experience

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Flask (Python)
  • Algorithms: MASB + QICE
  • Supported Formats: JPG, JPEG, PNG

Key Features

  • Advanced pixel scrambling using MASB
  • Quantum-chaotic key generation using QICE
  • Perfect recovery of original image after decryption
  • Beautiful modern UI with animated transitions
