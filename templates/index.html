IMAGE ENCRYPTOR


This project introduces a secure image encryption system using Multi-Agent Swarm Behavior (MASB) for pixel shuffling and Quantum-Inspired Chaotic Encoding (QICE) for highly random key generation. It ensures perfect image recovery during decryption while protecting data from unauthorized access. The system features a clean and interactive UI with drag-and-drop support and smooth animations for a seamless user experience.
